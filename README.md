## 旅游主题网站

保存有各个地区的信息

以及各地区中景点信息

以及相关景点的优惠套餐

### 项目运行

先去 setting.py 中配置MySQL数据库

安装需要的库文件

项目使用的Django版本为2.2

pip install -r requirements.txt

运行：

python manage.py runserver

访问 localhost：8000
